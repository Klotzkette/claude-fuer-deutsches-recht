# phishing_case_gate.py

Kleines Offline-Werkzeug für strukturierte Erstbewertung von Online-Banking-Phishing-Fällen.

Beispiel:

```bash
python phishing-vorfall-pruefer/scripts/phishing_case_gate.py --input testakten/phishing-vorfall-mayer-sparkasse-berlin/08_case_gate_input.json
```

Das Skript ersetzt keine rechtliche Prüfung. Es erzeugt eine reproduzierbare Ampel- und Aufgabenliste, damit die anwaltliche Bewertung nicht im Fließtext stecken bleibt.
