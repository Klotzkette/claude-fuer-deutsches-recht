# Sachbeihilfe und elan-Formalia

## Kaltstart

1. Wer stellt den Antrag und an welcher Institution?
2. Ist die wissenschaftliche Ausbildung abgeschlossen, regelmäßig Promotion?
3. Welche Module werden benötigt: Basismodul, Eigene Stelle, Vertretung, Rotationsstelle, Workshop, Mercator Fellow, Öffentlichkeitsarbeit?
4. Neuantrag, Fortsetzungsantrag oder Ergänzungsantrag?
5. Welche Vordruckversion ist aktuell?
6. Welche Anlagen: CV, Publikationsverzeichnis, Arbeitsprogramm, Finanzplan, Angebote, Kooperationsschreiben, Ethikvotum, Datenmanagement?

## Formaler Workflow

- DFG-Programmseite und Formularseite öffnen.
- elan-Programmauswahl und Fachzuordnung notieren.
- Seitenumfang und Gliederung gegen den aktuellen Leitfaden prüfen.
- Jede Anlage in eine Einreichungsmatrix eintragen.
- Fehlende institutionelle Erklärungen und Kooperationszusagen markieren.

## Output

| Abschnitt | Ergebnis |
| --- | --- |
| Programmschiene | Sachbeihilfe / Modulmix |
| Einreichungsreife | Grün / Gelb / Rot |
| Fehlende Anlagen | konkrete Liste |
| Risiken | formale Ablehnung, Nachforderung, Kürzungsrisiko |
| Nächster Skill | Text, Budget oder Reviewer-Red-Team |
