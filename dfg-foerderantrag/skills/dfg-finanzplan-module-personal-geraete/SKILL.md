# Finanzplan und Module

## Prinzip

Der Finanzplan ist ein wissenschaftliches Argument in Zahlen. Jede Position muss zeigen, welches Arbeitspaket ohne sie schwächer oder unmöglich wäre.

## Prüfpfad

1. Arbeitspakete aus der Projektbeschreibung übernehmen.
2. Ressourcen je Arbeitspaket zuordnen.
3. Personal: Qualifikation, Umfang, Dauer, Aufgabe.
4. Geräte: Notwendigkeit, Alleinstellungsbedarf, Mitnutzung, Angebote.
5. Reisen/Workshops: Forschungszweck, nicht Networking-Floskel.
6. Mercator Fellow/Kooperation: konkrete Rolle im Erkenntnisprozess.
7. Chancengleichheit/Öffentlichkeit: nur beantragen, wenn passend und begründet.

## Kürzungsabwehr

| Einwand | Gegenmittel |
| --- | --- |
| zu viel Personal | Arbeitspaket-Funktion und Auslastung zeigen |
| Gerät nicht nötig | Alternativen geprüft, Nutzungsplan belegen |
| Reisen diffus | konkrete Daten, Archive, Messungen, Kooperationen |
| Workshop dekorativ | Forschungsoutput definieren |

## Output

- Budgettabelle.
- Modulbegründung.
- Kürzungsrisiko-Ampel.
- Minimal-/Idealbudget-Vergleich.
